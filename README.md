# 🎨 이모티콘 메이커 (Imoticon Maker)

**누구나 쉽게, 말 한마디로 이모티콘 세트를 만드는 웹앱**

캐릭터와 상황을 자유롭게 설명하고 AI API 키를 연결하면, AI가 10~15개짜리 이모티콘 세트를 통째로 기획하고 그려 줍니다.

> 예시 입력:
> "여름에 더위에 시달리는 너구리 캐릭터를 기반으로 이모티콘 15개 만들어줘. 상황에 따라서 표정 변화와 그 감정을 잘 느낄 수 있도록 해주면 돼."

## ✨ 주요 기능

- **자연어 한 줄로 세트 기획** — AI가 캐릭터 시트(외형 고정)를 만들고, 일상 대화에서 실제로 쓰이는 감정/상황 10~15개 장면을 자동 기획합니다. 캐릭터 시트를 모든 이미지에 적용해 세트 전체의 캐릭터 일관성을 유지합니다.
- **캐릭터 참조 이미지 업로드** — 내 캐릭터 그림을 올리면 AI가 그 생김새를 분석해 캐릭터 시트를 만들고, 이미지 생성에도 참조로 전달해 일관성을 크게 높입니다. (Gemini: 이미지 입력, OpenAI: `images/edits`)
- **배경 투명화** — gpt-image-1은 API 네이티브 투명 배경, 그 외 모델은 가장자리 flood-fill 알고리즘으로 흰 배경만 정확히 제거합니다 (캐릭터 안의 흰 부분은 보존).
- **움직이는 이모티콘 (APNG + GIF)** — 컷마다 4프레임 동작을 AI가 기획하고, 1번 프레임을 참조 이미지로 삼아 2~4번 프레임을 그려 프레임 간 일관성을 유지합니다. 브라우저에서 직접 APNG와 GIF로 인코딩합니다 (외부 라이브러리 없음).
- **AI 제공자 선택 (BYOK)** — 내 API 키로 직접 호출합니다.
  - Google Gemini: 기획 `gemini-2.5-flash` + 이미지 `gemini-2.5-flash-image` (무료 할당량 있음, 추천)
  - OpenAI: 기획 `gpt-4o-mini` + 이미지 `gpt-image-1` (권한 없으면 자동으로 `dall-e-3` 대체)
- **스타일 선택** — 귀여운 스티커풍 / 파스텔 / 카툰 / 라인 드로잉 / 수채화
- **한글 문구 옵션** — 이모티콘 안에 짧은 한글 문구("더워…", "최고!") 포함 가능
- **개별 재생성** — 마음에 안 드는 컷만 골라서 다시 그리기
- **다운로드** — 개별 PNG 저장 또는 전체 ZIP 저장, 카카오 이모티콘 규격(360×360) / 라인·텔레그램(512×512) 자동 리사이즈

## 🚀 사용 방법

빌드나 설치가 필요 없는 순수 정적 웹앱입니다.

```bash
# 아무 정적 서버로 실행
npx serve .
# 또는
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속 후:

1. **AI 연결하기** — 제공자를 고르고 API 키를 붙여넣기
   - Gemini 키 발급: https://aistudio.google.com/apikey
   - OpenAI 키 발급: https://platform.openai.com/api-keys
2. **설명 입력** — 만들고 싶은 캐릭터와 컨셉을 자유롭게 작성 (예시 칩 클릭으로 빠른 시작)
3. **✨ 이모티콘 만들기** 클릭 → 진행 상황을 보며 대기
4. 완성된 그리드에서 저장/재생성, 또는 **전체 ZIP 다운로드**

GitHub Pages에 올리면 URL 접속만으로 바로 사용할 수 있습니다.

## 🔒 개인정보 / 보안

- API 키와 생성된 이미지는 **서버로 전송되지 않습니다.** 브라우저에서 AI API로 직접 호출합니다.
- "키 저장"을 체크하면 키가 내 브라우저의 `localStorage`에만 저장됩니다. 공용 PC에서는 체크하지 마세요.

## 🗂 프로젝트 구조

```
index.html          # UI (단일 페이지)
css/style.css       # 스타일
js/app.js           # 메인 로직 (파이프라인, 카드 렌더링, 다운로드)
js/providers.js     # AI 어댑터 (Gemini / OpenAI) + 프롬프트 빌더
js/imageproc.js     # 이미지 유틸 (리사이즈, 배경 투명화, 업로드 정규화)
js/apng.js          # 의존성 없는 APNG 조립기 (움직이는 PNG)
js/gif.js           # 의존성 없는 GIF 인코더 (미디언컷 팔레트 + LZW)
js/zip.js           # 의존성 없는 경량 ZIP 생성기
```

## 🧭 동작 원리

```
사용자 설명 (+ 캐릭터 참조 이미지)
   │
   ▼
[1단계: 기획]  텍스트/비전 모델이 JSON으로 반환
   ├── character: 캐릭터 외형 고정 묘사 (참조 이미지가 있으면 그 캐릭터 분석)
   └── ideas[]: 장면별 { 제목, 감정/포즈 묘사, 한글 문구, (움직임이면) 프레임 4개 }
   │
   ▼
[2단계: 그리기]  장면마다 이미지 모델 호출 (동시 2개, 429 자동 재시도)
   ├── 정지: 1장 생성
   └── 움직임: 프레임 1 생성 → 그 이미지를 참조로 프레임 2~4 생성 (일관성 유지)
   │
   ▼
[3단계: 후처리]  배경 투명화 (API 네이티브 또는 flood-fill)
   │
   ▼
[4단계: 내보내기]  캔버스 리사이즈 → PNG / APNG / GIF / ZIP
```
